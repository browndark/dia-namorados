💕 Guia: Publicar no GitHub Pages

Passo-a-passo para deixar seu site ao vivo na internet

1Criar Repositório no GitHub

Vá para https://github.com e faça login com sua conta (se não tiver, crie uma gratuitamente)

  1. Clique no + (sinal de mais) no canto superior direito
  2. Selecione "New repository"
  3. Preencha os campos:
    • Repository name: dia-namorados
    • Description (opcional): "Um site de presente para Thiane no Dia dos Namorados"
    • Privacidade: Selecione "Public" (precisa ser público para GitHub Pages funcionar)
  4. ⚠️ NÃO MARQUE "Initialize this repository with a README"
  5. Clique em "Create repository"

✅ Seu repositório foi criado!

2Fazer Upload dos Arquivos

Você verá uma página vazia do repositório. Agora vamos adicionar os arquivos:

  1. Clique em "Add file" (botão verde)
  2. Selecione "Upload files"
  3. Arraste os arquivos da pasta docs para a área de upload:
    • ✅ index.html
    • ✅ styles.css
    • ✅ script.js
    • ✅ Pasta assets (com imagens e música)
  4. Clique em "Commit changes" para confirmar
💡 Dica: Se preferir, você pode criar as pastas e arquivos um a um usando "Add file" > "Create new file". Mas upload em lote é mais rápido!

3Ativar GitHub Pages

Agora vamos publicar o site:

  1. Na página do repositório, clique em "Settings" (ícone de engrenagem, no topo)
  2. No menu esquerdo, clique em "Pages"
  3. Em "Source", mude para:
    • Branch: main
    • Folder: / (root)
  4. Clique em "Save"
  5. O GitHub criará seu site! Aguarde uns 2-3 minutos...
Se aparecer um erro sobre "index.html não encontrado", certifique-se que você fez upload do arquivo index.html diretamente na raiz do repositório.

4Encontrar sua URL

Volte para a página de Settings > Pages e procure por uma mensagem como:

Your site is live at: https://seu-username.github.io/dia-namorados/

Essa é a URL do seu site! 🎉 Você pode:

  • ✅ Compartilhar com Thiane
  • ✅ Enviar pelo WhatsApp, email, etc
  • ✅ Publicar nas redes sociais
  • ✅ Adicionar aos favoritos do navegador
O site ficará online para sempre (ou enquanto o repositório existir)!

5Customizações Futuras (Opcional)

Se quiser mudar algo depois, é fácil:

  • Mudar a mensagem: Edite index.html no repositório
  • Mudar as cores: Edite styles.css
  • Adicionar mais fotos: Adicione as imagens em assets/images/ e edite script.js

Cada vez que você faz alterações, o GitHub atualiza automaticamente o site em alguns segundos.

❓Dúvidas Comuns

P: Quanto tempo leva para ficar pronto?

R: Geralmente 2-3 minutos, mas pode levar até 10 minutos em alguns casos.

P: A música não toca no celular!

R: Alguns navegadores mobile bloqueiam autoplay. A pessoa pode clicar no ícone 🎵 para iniciar.

P: Posso usar um domínio próprio?

R: Sim! Em Settings > Pages > Custom domain, adicione seu domínio.

P: O site é gratuito?

R: Totalmente! GitHub Pages é 100% gratuito. Não há limites de tempo ou visitantes.

P: Posso manter o site privado?

R: Não, GitHub Pages requer repositório público. Mas qualquer pessoa precisa da URL para acessar.
